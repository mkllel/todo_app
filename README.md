# 할 일 목록 앱

이 프로젝트는 React로 만들어진 할 일 목록 애플리케이션입니다. 사용자는 할 일을 추가, 완료 표시, 삭제할 수 있으며, 페이지를 새로고침해도 할 일 목록이 유지됩니다. 또한, 매일 자정에 할 일 목록이 자동으로 초기화되도록 하였습니다.

## 기능

- 새로운 할 일 추가
- 할 일을 완료로 표시
- 할 일 삭제
- 로컬 스토리지를 사용한 데이터 저장
- 매일 자정 할 일 목록 초기화
- 반응형 디자인
